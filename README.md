# ft_select

Ce projet a pour but de vous faire coder un petit programme en termcaps qui
permettra de selectionner un ensemble de choix parmi une liste et de le retourner Ã 
votre shell.

Compile ft_select
```

```

Test:
```
ls -l `./ft_select *`
```
or just 
```
make test
```
